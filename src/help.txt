MENU Auto.Ria.ua App

 Using our program you can:

  !!! Enter the desired item in the command line (from 1 to 8) !!!

   1 - Show list cars
     In this item, you can see list cars
	 
   2 - Show list buses
     In this item, you can see list buses
	 
   3 - Search (by model)
     In this item, you can see list cars by model.

   4 - Search (by model and year)
     In this item, you can see list cars by model and year.
	 
   5 - Search (by price and year)
     In this item, you can see list cars by price and year.

   6 - Filter Cars With Insureance
     In this item, you can see list cars by price and year.

   7 - Count cars by model
     In this item, you can see count cars by model.

   8 - Help
     In this item, you can see support telephone number.

   0 - Quit and file close
     Select this option if you want to close the program and exit it.
